Web application for creating **.ico** and **.icns** files from image.

Try demo [**here**](https://unicon-icon.herokuapp.com).

## Running locally <br/>

Install needed requirements: <br/>
`$ pip3 install requirements.txt`<br/>

And run next command:<br/>
`$ python3 app.py`<br/>
